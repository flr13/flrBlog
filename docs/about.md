# 关于 FLR技术馆

## 👋 欢迎来到 FLR技术馆

这里是一个热爱编程分享的前端开发技术平台，致力于记录前端开发道路上的探索与实践，分享技术心得与工程经验。

## 📚 关于我们

FLR技术馆是一个专注于前端技术的学习与分享平台，由热爱编程的前端开发者创建和维护。我们致力于：

- 分享前端开发的技术知识和最佳实践
- 探讨前端工程化和性能优化
- 交流前端框架和工具的使用经验
- 构建一个活跃的前端技术社区

## 🛠️ 技术栈

- **前端框架**：Vue、React
- **开发语言**：JavaScript、TypeScript
- **构建工具**：Vite、Webpack
- **样式方案**：CSS3、SCSS
- **代码质量**：ESLint、Prettier
- **版本控制**：Git

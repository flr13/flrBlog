# Python 零基础入门核心教程（MD 版）

## 教程说明

本教程专为**零基础小白**设计，避开复杂概念，从环境搭建到基础语法、实战案例，层层递进，所有代码可直接运行，快速掌握 Python 核心基础，为后续数据分析、AI / 爬虫等方向打基础。

**学习原则**：多敲代码＞死记硬背，每节代码至少手动运行 1 次，理解逻辑比记语法更重要。

## 一、环境搭建（最简易版，新手首选）

Python 是**跨平台语言**，支持 Windows/Mac/Linux，推荐用「Python 解释器 + VS Code」组合，轻量、易操作、适配所有场景。

### 1. 安装 Python 解释器



1. 官网下载：[Python 官方下载](https://www.python.org/downloads/)，选择**3.9 + 版本**（推荐 3.10/3.11，稳定无兼容问题）；

2. 安装关键：**勾选 Add Python to PATH**（添加到系统环境变量，否则无法在命令行运行 Python），然后一路下一步；

3. 验证安装：打开电脑「命令提示符（CMD）/ 终端」，输入`python --version`，显示版本号（如 Python 3.10.12）即安装成功。

### 2. 安装 VS Code（代码编辑器）



1. 官网下载：[VS Code 官方下载](https://code.visualstudio.com/Download)，免费安装，一路下一步；

2. 安装 Python 插件：打开 VS Code，左侧点击「扩展」，搜索**Python**，安装微软官方的 Python 插件（带蓝色对勾），自动适配语法高亮、代码提示、运行功能。

### 3. 第一个 Python 程序（Hello World）



1. VS Code 中新建文件夹，新建文件命名为`hello.py`（**后缀必须是.py**，代表 Python 文件）；

2. 输入代码：



```
\# 打印输出，Python中最基础的语句

print("Hello Python! 你好，Python！")
```



1. 运行代码：右键点击文件 → 「在终端中运行 Python 文件」，终端显示`Hello Python! 你好，Python！`，即成功运行。

## 二、Python 基础语法（核心必学，极简版）

Python 语法**无分号、靠缩进**（4 个空格 / 1 个 Tab），贴近自然语言，以下是零基础必掌握的核心知识点，每个知识点配可运行代码。

### 1. 注释（代码说明，不执行）

注释是给自己 / 别人看的代码说明，Python 不会执行注释内容，分两种：



```
\# 单行注释：以#开头，注释一行内容（最常用）

print("这是执行的代码")  # 行尾也可以加单行注释

"""

多行注释：用三个双引号/单引号包裹，注释多行内容

一般用于文件开头、函数/类的说明

"""

'''

这也是多行注释，和双引号用法一致

'''

print("多行注释不影响代码执行")
```

### 2. 变量与数据类型（存储数据的容器）

#### （1）变量定义：**直接赋值，无需声明类型**（Python 自动识别）



```python
\# 变量名 = 变量值，等号=是赋值符号

name = "小明"  # 字符串类型

age = 18       # 整数类型

height = 175.5 # 浮点数类型（小数）

is\_student = True # 布尔类型（只有True/False两个值，代表真/假）
```

**变量名规则**：字母 / 数字 / 下划线组成，不能以数字开头，区分大小写（如`Age`和`age`是两个变量），不用中文 / 特殊符号。

#### （2）常用数据类型（4 种基础类型，必记）



| 类型名称 | 英文    | 说明              | 示例                                   |
| ---- | ----- | --------------- | ------------------------------------ |
| 字符串  | str   | 文本内容，用单 / 双引号包裹 | `name = "Python"`、`msg = '学习Python'` |
| 整数   | int   | 正负数 / 0，无小数     | `num = 100`、`score = -60`            |
| 浮点数  | float | 小数，包含正负数        | `price = 9.9`、`weight = -2.5`        |
| 布尔值  | bool  | 只有 True/False   | `is_ok = True`、`is_error = False`    |

#### （3）类型查看与转换



```python
\# type(变量)：查看变量的数据类型

a = 18

print(type(a))  # 输出\<class 'int'>，说明a是整数类型

b = "18"

print(type(b))  # 输出\<class 'str'>，说明b是字符串类型

\# 类型转换：类型名(变量)，常用int/str/float互转

c = int(b)      # 字符串转整数，b是"18"，转成18

print(c + 2)    # 输出20，整数可以做加法

d = str(a)      # 整数转字符串，a是18，转成"18"

print(d + "岁") # 输出18岁，字符串可以拼接
```

### 3. 输入与输出（和用户交互）

#### （1）输出：`print()`，打印内容到终端（最常用）



```python
\# 直接输出单个内容

print(18)

\# 输出多个内容，用逗号分隔，自动空格

print("姓名：", "小明", "年龄：", 18)

\# 格式化输出（推荐，更美观）：f-string，字符串前加f，变量用{}包裹

name = "小红"

age = 17

print(f"我的名字是{name}，今年{age}岁") # 输出：我的名字是小红，今年17岁
```

#### （2）输入：`input()`，获取用户在终端输入的内容



```python
\# input("提示语")，用户输入的内容\*\*默认是字符串类型\*\*

name = input("请输入你的名字：")

age = int(input("请输入你的年龄：")) # 输入年龄转成整数

print(f"你好{name}，明年你{age+1}岁")
```

**运行效果**：终端显示「请输入你的名字：」，输入后按回车，再提示输入年龄，最后输出结果，实现和用户的简单交互。

### 4. 运算符（做计算 / 判断，4 类核心）

#### （1）算术运算符（做数学计算，最常用）



```python
a = 10

b = 3

print(a + b)  # 加法，13

print(a - b)  # 减法，7

print(a \* b)  # 乘法，30

print(a / b)  # 除法，3.3333333333333335

print(a // b) # 整除，取商，3

print(a % b)  # 取余，取余数，1

print(a \*\* b) # 幂运算，a的b次方，1000
```

#### （2）赋值运算符（给变量赋值，+=/-= 最常用）



```python
x = 5

x += 2  # 等价于x = x + 2，x变成7

x -= 3  # 等价于x = x - 3，x变成4

x \*= 2  # 等价于x = x \* 2，x变成8

print(x)
```

#### （3）比较运算符（判断大小，返回布尔值 True/False）



```python
a = 10

b = 5

print(a > b)   # 大于，True

print(a < b)   # 小于，False

print(a == b)  # 等于，判断两个值是否相等，False（注意：是==，不是=）

print(a != b)  # 不等于，True

print(a >= b)  # 大于等于，True

print(a <= b)  # 小于等于，False
```

#### （4）逻辑运算符（多条件判断，and/or/not）



```python
\# and：且，所有条件都为True，结果才是True

print(10>5 and 8<10) # True

\# or：或，只要有一个条件为True，结果就是True

print(10<5 or 8<10) # True

\# not：非，取反，True变False，False变True

print(not 10>5) # False
```

### 5. 条件判断（根据条件执行不同代码，if-else）

核心：**满足条件执行 A，不满足执行 B**，靠缩进区分代码块，格式固定，必记！

#### （1）基础版：if



```python
age = 18

if age >= 18:

&#x20;   print("你是成年人")  # 条件满足（age>=18），执行这行（注意缩进）
```

#### （2）常用版：if-else



```
age = int(input("请输入你的年龄："))

if age >= 18:

&#x20;   print("你是成年人，可进入")

else:

&#x20;   print("你是未成年人，禁止进入") # 条件不满足，执行这行
```

#### （3）多条件版：if-elif-else



```
score = int(input("请输入你的分数："))

if score >= 90:

&#x20;   print("成绩优秀")

elif score >= 80:

&#x20;   print("成绩良好")

elif score >= 60:

&#x20;   print("成绩及格")

else:

&#x20;   print("成绩不及格，需要补考")

\# 注意：elif可以有多个，else可选，条件满足后直接跳出，不会执行后续判断
```

### 6. 循环（重复执行代码，for/while，必学 for 循环）

#### （1）for 循环（遍历序列，最常用，如遍历数字 / 字符串 / 列表）

**经典用法：遍历数字范围**`range()`



```
\# range(n)：生成0到n-1的整数序列

for i in range(5):

&#x20;   print(i) # 输出0,1,2,3,4（换行）

\# range(a,b)：生成a到b-1的整数序列

for i in range(1,6):

&#x20;   print(f"第{i}次循环") # 输出第1到第5次循环

\# range(a,b,step)：步长step，每次加step

for i in range(1,10,2):

&#x20;   print(i) # 输出1,3,5,7,9（步长2，隔一个数）
```

#### （2）while 循环（条件满足就循环，直到条件不满足）



```
\# 格式：while 条件: 循环代码

count = 1

while count <= 3:

&#x20;   print(f"第{count}次执行while循环")

&#x20;   count += 1 # 必须让条件最终不满足，否则死循环！

\# 输出：第1/2/3次执行while循环
```

**注意**：while 循环必须加「变量自增 / 自减」，否则条件永远满足，进入**死循环**（按 Ctrl+C 终止）。

#### （3）循环终止：break/continue（可选）



```
\# break：直接终止整个循环

for i in range(1,6):

&#x20;   if i == 3:

&#x20;       break

&#x20;   print(i) # 输出1,2（i=3时终止循环）

\# continue：跳过当前次循环，继续下一次

for i in range(1,6):

&#x20;   if i == 3:

&#x20;       continue

&#x20;   print(i) # 输出1,2,4,5（跳过i=3）
```

### 7. 列表（存储多个数据的容器，最常用）

Python 中最基础的**有序可变**容器，可存储不同类型数据，用`[]`定义。



```
\# 定义列表：变量 = \[元素1, 元素2, 元素3,...]

student = \["小明", 18, 175.5, True]

\# 访问列表元素：列表\[索引]，索引从0开始！

print(student\[0]) # 输出第一个元素：小明

print(student\[1]) # 输出第二个元素：18

\# 常用操作：增/删/查/改

student.append("男") # 增加：在列表末尾加元素

student\[0] = "小红"  # 修改：通过索引修改元素

student.remove(18)   # 删除：删除指定元素

print(student)       # 输出修改后的列表：\['小红', 175.5, True, '男']

\# 遍历列表（最常用，用for循环）

fruits = \["苹果", "香蕉", "橘子"]

for fruit in fruits:

&#x20;   print(f"水果：{fruit}") # 依次输出每个水果
```

## 三、函数（封装重复代码，复用性高）

函数是**封装好的可重复执行的代码块**，避免重复写相同代码，Python 内置了大量函数（如`print()`/`input()`），也可以自定义函数。

### 1. 自定义函数（核心格式）



```
\# 定义函数：def 函数名(参数): 函数体 return 返回值

def add(a, b):

&#x20;   """求两个数的和"""

&#x20;   result = a + b

&#x20;   return result # 返回计算结果，可选

\# 调用函数：函数名(参数)，获取返回值

sum1 = add(3,5)

sum2 = add(10,20)

print(sum1, sum2) # 输出8 30

\# 无参数无返回值的函数（简单）

def say\_hello():

&#x20;   print("你好，Python函数！")

say\_hello() # 调用：直接输出内容
```

**核心**：定义时用`def`，调用时直接写函数名，参数可多可少，返回值`return`可选（无 return 则返回`None`）。

### 2. 内置函数（Python 自带，直接用，必记几个）

Python 无需定义，直接调用的函数，日常开发高频使用：



* `print()`：输出

* `input()`：输入

* `type()`：查看数据类型

* `len()`：获取长度（列表 / 字符串等）

* `int()/str()/float()`：类型转换



```
s = "Python"

print(len(s)) # 输出6，获取字符串长度

lst = \[1,2,3]

print(len(lst)) # 输出3，获取列表长度
```

## 六、学习技巧（零基础避坑）



1. **多敲代码**：看 10 遍不如敲 1 遍，代码报错是正常的，解决报错的过程就是进步；

2. **善用报错信息**：终端报错时，看最后一行的错误提示，百度错误信息即可找到解决方案；

3. **找简单项目练手**：基础掌握后，做小型实战项目（如批量重命名文件、爬取简单网页），不要只学语法；

4. **不用死记硬背**：语法记不住可以查，核心是理解**逻辑**（条件判断 / 循环 / 函数的作用）；

5. **用官方文档**：Python 官方文档（[Python Docs](https://docs.python.org/zh-cn/3/)）是最权威的资料，不懂就查。

## 七、常见问题解决



1. **运行 Python 提示 “不是内部或外部命令”**：安装 Python 时没勾选「Add Python to PATH」，重新安装并勾选，或手动配置环境变量；

2. **代码缩进报错**：Python 缩进必须统一（要么全用 4 个空格，要么全用 Tab），不能混合使用；

3. **中文乱码**：VS Code 中新建文件时，选择编码为**UTF-8**（默认就是，无需修改）；

4. **模块导入报错**：如`import random`报错，一般是 Python 安装损坏，重新安装即可。
